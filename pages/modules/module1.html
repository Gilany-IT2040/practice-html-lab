Absolute File Paths

An absolute file path is the full URL to a file or resource. It includes the protocol, domain, and the path to the file. For example, https://www.example.com/images/logo.png is an absolute file path.

Generally, you only want to use absolute file paths if you're referencing a file or a resource or a webpage that's on your own website. a link to another website for example

You never want to use absolute file paths when referencing files or resources on your own computer. This is because we won't have access to the files on your computer when we view your website.
